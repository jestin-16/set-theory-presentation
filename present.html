Use of Sets in Data Structures & Programming

Sets used to store unique elements (no duplicates).

Examples: Python set(), Java HashSet.

Helps in faster searching and filtering.

# Python example
my_set = {1, 2, 3, 4, 5}
my_set.add(3) # No duplicate added
my_set.add(6)
print(my_set) # {1, 2, 3, 4, 5, 6}

Sets in Databases

SQL operations like UNION, INTERSECT, EXCEPT come from set theory.

Databases treat query results as mathematical sets.

Sets in Computer Logic & AI

Logic circuits use set operations for decision-making.

In AI, sets help group features, filter data, and classify information.
